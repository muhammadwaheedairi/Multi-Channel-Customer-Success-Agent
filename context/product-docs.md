# TechCorp Product Documentation

## Authentication
- Login at app.techcorp.io with email + password
- Reset password: Settings → Security → Reset Password
- 2FA: Settings → Security → Enable 2FA (Google Authenticator)
- SSO available on Enterprise plan only

## Projects
- Create: Dashboard → New Project → Add name, team, deadline
- Invite team: Project → Settings → Members → Invite by email
- Archive: Project → Settings → Archive Project

## Billing
- View invoices: Settings → Billing → Invoice History
- Update card: Settings → Billing → Payment Method
- Cancel: Settings → Billing → Cancel Subscription
- Refunds: Handled by human billing team only

## Integrations
- Slack: Settings → Integrations → Slack → Connect
- Google: Settings → Integrations → Google Workspace
- API Key: Settings → Developer → API Keys

## Common Issues
- Slow loading: Clear cache, try incognito, check status.techcorp.io
- Can't invite member: Check if email already registered
- Export data: Settings → Data → Export (CSV/JSON)
